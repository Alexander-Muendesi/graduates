## Basic Scripts

These are essential scripts to get started:

```sh
yarn run start
yarn run start:client
yarn run start:api
yarn run start:db
yarn run prisma:studio
yarn run prisma:generate
yarn run prisma:migrate:dev
yarn run build
yarn run lint
yarn run test
```
